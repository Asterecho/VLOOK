[+] Markdown 是什么？

  • 2004 年，John Gruber 创造了 Markdown，一种专门针对网络写作的文本标记语言。使用 Markdown，你只需在写作过程中插入少量的标记符号，就能很轻松地进行排版（例如设置标题、加粗、列表、引用等）；
  • Markdown 文档以纯文本格式存储，这意味着，它们可以用几乎任一种文本编辑器打开。同时，又能通过 Markdown 编辑器导出为带排版的富文本文档、HTML 网页等等。纯粹、简洁、易用、灵活，都是人们喜欢 Markdown 的原因；
  • 目前 Markdown 的标准化项目是 CommonMark。

[+] 60 秒学会、10 分钟深入学习 Markdown 语法：

  1. 推荐 Github Flavored Markdown (GFM) 语法参考：Typora 目前采用该标准 详细；
  2. 标准化 CommonMark 语法参考：60 秒学会 Markdown 语法、10 分钟深入学习 Markdown。
