# 特性
- 自动生成大纲结构（基于`[TOC]`），以及分级序号<br>
- 支持钉住/收起大纲导航、前进/后退<br>
- 自动根据文档标题生成水印背景<br>
- 检查无效的页内链接、重复标题<br>
- 自适应设备屏幕大小显示<br>
- 美观、细腻、动态的页面样式<br>

# 使用
使用指定的theme主题，并通过Typora导出的HTML文件。<br>
1. 导出的MD文档必须要添加`[TOC]`标签；<br>
2. 导出后将大纲代码直接粘贴到导出的HTML文件的`</body>`前即可，详见示例。
